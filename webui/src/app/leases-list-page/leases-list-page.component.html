This page displays a list of leases in the network. Leases are retrieved at regular intervals, which can be configured in the Settings panel under the Configuration menu in the top bar. Kea must be configured to use a memfile, not an SQL database, for lease storage.

By default, the agents do not attempt to collect lease data from the Kea servers for this feature. To instruct them to do so, please start each agent with the flag --enable-lease-tracking or with the environment variable STORK_AGENT_ENABLE_LEASE_TRACKING=true. For more information about setting up this feature, consult the Leases List section of the Stork Administrator Reference Manual. The troubleshooting section of the ARM contains additional information about resolving issues with enabling this feature.
